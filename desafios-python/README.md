# 🐍 Desafios em Python — FATEC Linguagem de Programação I

> Projeto de portfólio com 6 desafios de programação resolvidos em Python,
> seguindo a metodologia **Procedural → Modular** da disciplina.

---

## 📌 Sobre o Projeto

Este repositório contém as soluções para os 6 primeiros desafios da **Lista de Exercícios: Funções** da disciplina **Linguagem de Programação I** (FATEC).

Cada desafio é resolvido em **duas versões**:

| Versão | Abordagem |
|--------|-----------|
| **Versão 1 — Procedural** | Código sequencial, sem funções auxiliares |
| **Versão 2 — Modular** | Código organizado em funções reutilizáveis |

O objetivo é demonstrar na prática a evolução natural do código: do simples ao modular.

## ▶️ Como Executar o Projeto

### Pré-requisitos

- Python 3.10 ou superior instalado
- Verificar com: `python --version` ou `python3 --version`

### Passo a passo

**1. Clone o repositório**

```bash
git clone https://github.com/SEU_USUARIO/desafios-python.git
cd desafios-python
```

**2. Entre na pasta do desafio desejado**

```bash
cd desafio1   # ou desafio2, desafio3, etc.
```

**3. Execute o programa**

```bash
python main.py
```

**4. Siga as instruções no terminal**

Cada programa irá perguntar qual versão executar (Procedural ou Modular) e depois solicitará os dados de entrada.
