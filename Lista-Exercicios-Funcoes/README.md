# Lista de Exercícios — Funções
**Disciplina:** Linguagem de Programação I — FATEC  
**Tema:** Funções e Modularização

---

## Estrutura do Projeto

Cada exercício contém **duas versões** implementadas em **três linguagens**:

| Versão | Descrição |
|--------|-----------|
| Versão 1 | Procedural — código sequencial, sem funções auxiliares |
| Versão 2 | Modular com Funções — lógica isolada em funções reutilizáveis |

| Linguagem | Pasta | Arquivo |
|-----------|-------|---------|
| Python | `python_logic/` | `main.py` |
| C++ | `cpp_logic/` | `main.cpp` |
| Java | `java_logic/` | `Main.java` |

---

## Como Executar

### Python
```bash
python3 python_logic/main.py
```

### C++
```bash
g++ cpp_logic/main.cpp -o main && ./main
```

### Java
```bash
javac java_logic/Main.java && java -cp java_logic Main
```
